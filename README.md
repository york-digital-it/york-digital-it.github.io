## üìñ Getting Started

Want to run the site locally?

1. **Clone the repository:**
   ```sh
   git clone https://github.com/york-digital-it/york-digital-it.github.io.git
   cd york-digital-it.github.io
   ```

2. **If using Jekyll:**
   Install Ruby, Bundler, and run:
   ```sh
   bundle install
   bundle exec jekyll serve
   ```
   Then open [http://localhost:4000](http://localhost:4000) in your browser.

3. **If using plain HTML/CSS/JS:**
   Just open `index.html` in your web browser.
